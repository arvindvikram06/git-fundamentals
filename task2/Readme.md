# Git Fundamentals - Task 2
## Stage, Commit, Push, and Ignore
 
---
 
## Objective
- Stage new files using `git add`
- Commit changes with a descriptive message
- Add a `.gitignore` file to exclude folders
- Push commits to GitHub
- Verify commit history
 
---
 
## Steps Performed
 
### Check Repository Status
```
git status
```
 
---
 
### Stage All Files
```
git add .
```
 
---
 
### Commit with a Message
```
git commit -m "completed task2 (added gitignore to ignore envfolder)"
```
 
---
 
### Push to GitHub
```
git push origin master
```
 
---
 
### Verify Commit History
```
git log --oneline
```
